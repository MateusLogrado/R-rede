Conceito de um servidor

O servidor é um local que pode amazenar informações,
administrar arquivos/informações e atender ao cliente, fornecendo oque ele necessida

Oque é a Apache?

A apache é um servidor de codigo aberto capaz de armazenar sites,
tendo 46% dos site armazenados nele

Oque é o Tomcat?

O Tomcat foi desenvolvido pela Apache, que permite a execução de aplicações na web, focado na linguagem JAVA
