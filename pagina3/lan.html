Oque a LAN?

São dispositivos proximos fisicamente e que estão usando conectores como roteadores

Oque a MAN?

A MAN é uma rede computadores conectadas dentro de uma cidade

Oque a WAN?

A WAN é uma rede de computadores que abrange uma grande escala geografica

Oque é PAN?

È uma rede de área pessoal que conecta dispositivos eletrocinocos dentro
de uma area especifica, normalmente sendo tamanho de uma casa
