Conceito WEB

A palavra WEb significa hoje em dia um
sistema de informações ligadas por meio das mídias.


Conceito de WWW

O WWW é uma sigla para World Wide Web, que pode ser traduzido
para "teia do tamanho do mundo"
Ele é um modelo de um sistema interligado de informações e arquivos.

Conceito de W3C

A W3C é uma organização de varios paises, privados, órgãos governamentais e associações independentes.
Que atuam para fazer protcolos e padronizações para a rede mundial de computadores ou WWW
